# WindTurbine_Map

### 緣由
由於台灣能源的轉型，推動離岸風電在台的開發。
再加上全世界前20大風場台灣海峽就有16個，
所以吸引許多國外的開發商前來投資，
同時也帶動水下基礎與風機零組件國產化的發展。

近期推動第三階段風場區塊開發，所以繪製以下的潛力場址的互動式地圖。

### 座標
這次基於TWD97二度分帶坐標值轉換為經緯度座標所繪製。
